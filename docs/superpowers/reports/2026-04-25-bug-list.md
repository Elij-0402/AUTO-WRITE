## Bug 清单

### BUG-001: 离线优先承诺被破坏 — 未登录用户被强制重定向到 /auth

- **模块**: 全局 / 认证
- **严重**: P0
- **状态**: fixed
- **来源**: e2e
- **修复 commit**: `8bdbbdb` — 移除 `src/proxy.ts` 中强制重定向逻辑

**复现**：（修复前）
1. 启动 `pnpm dev`
2. 用未登录浏览器访问 `http://localhost:3000/`
3. 实际：被 `src/proxy.ts` 重定向到 `/auth`

**期望**：未登录用户可以正常使用所有本地功能；只有用户主动启用云同步才进入 /auth。

### BUG-005: TypeScript 类型错误阻塞生产构建

- **模块**: 构建 / AI
- **严重**: P0
- **状态**: fixed
- **来源**: build
